5. **Click**: "Create repository"
6. **Copy the URL** shown (looks like: `https://github.com/YOUR_USERNAME/bingo-mini-app.git`)

### STEP 2: Push Your Code (1 minute)

Open Command Prompt in your project folder and run these 3 commands:

```bash
# Command 1: Add GitHub remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/bingo-mini-app.git

# Command 2: Set branch name
git branch -M main

# Command 3: Push code
git push -u origin main
```

**Wait for upload to complete** (about 30 seconds)

### STEP 3: Deploy to Render (5 minutes)

1. **Go to**: https://render.com
2. **Sign up** with GitHub
3. **Click**: "New +" → "Web Service"
4. **Connect**: `bingo-mini-app` repository
5. **Configure**:
   - Build Command: `npm run render-build`
   - Start Command: `npm run render-start`
6. **Add Environment Variables** (copy from `DEPLOYMENT_QUICK_REFERENCE.md`)
7. **Click**: "Create Web Service"
8. **Wait**: 5-10 minutes for deployment
