📋 WHAT YOU NEED TO DO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Configure Environment Variables (15 minutes)
   ├─ Create Aiven MySQL database (free tier)
   ├─ Create Telegram bot via @BotFather
   ├─ Create Cloudinary account (free tier)
   └─ Edit .env and client/.env files

2. Deploy Database Schema
   └─ Run: mysql -h your-host.aivencloud.com -u avnadmin -p < server/database/schema.sql

3. Start the Application
   ├─ Windows: start-dev.bat
   └─ Manual: npm start (backend) + cd client && npm start (frontend)

4. Test Everything
   ├─ Backend: http://localhost:3001/health
   ├─ Frontend: http://localhost:3000
   └─ Telegram: Configure bot menu button

🎮 GAME FLOW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. User deposits funds
   └─ Uploads payment screenshot via Cloudinary

2. Admin approves deposit
   └─ Balance added to main wallet

3. User transfers to play wallet
   └─ Funds ready for games

4. User joins game
   └─ Bet deducted, receives Bingo card

5. Game starts (admin)
   └─ Numbers drawn every 7 seconds

6. User wins
   └─ Claims BINGO, prize sent to main wallet
