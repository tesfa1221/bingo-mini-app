┌──────────────────────────────────────────────────────────────────────────────┐
│ 🎯 HOW IT WORKS                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

1. USER SENDS /start
   └─> Bot shows welcome message with buttons

2. USER CLICKS "Register / Update Profile"
   ├─> If already registered: Shows account status
   └─> If new user: Requests contact sharing

3. USER CLICKS "Share My Contact"
   └─> Telegram sends contact to bot

4. BOT RECEIVES CONTACT
   ├─> Verifies it's user's own contact (not someone else's)
   ├─> Checks if phone number already registered
   ├─> Stores: telegram_id, username, phone_number, first_name, last_name
   └─> Sends success message with account details

5. USER IS NOW REGISTERED
   └─> Can deposit, play, and withdraw
