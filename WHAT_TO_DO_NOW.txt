📋 WHAT YOU NEED TO DO (3 SIMPLE STEPS):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Get Your Credentials (15 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A) Aiven MySQL Database (5 min)
   1. Go to: https://aiven.io
   2. Sign up (free tier)
   3. Create MySQL service
   4. Copy: Host, User, Password

B) Telegram Bot (3 min)
   1. Open Telegram
   2. Search: @BotFather
   3. Send: /newbot
   4. Copy bot token
   5. Search: @userinfobot
   6. Copy your ID

C) Cloudinary (5 min)
   1. Go to: https://cloudinary.com
   2. Sign up (free tier)
   3. Copy: Cloud Name, API Key, API Secret
   4. Settings → Upload → Add preset
   5. Name: bingo_deposits
   6. Mode: Unsigned


STEP 2: Configure Environment Files (2 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Edit file: .env
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Replace these values with your actual credentials:

DB_HOST=your-aiven-host.aivencloud.com
DB_PASSWORD=your-password
TELEGRAM_BOT_TOKEN=your-bot-token
CLOUDINARY_CLOUD_NAME=your-cloud-name
CLOUDINARY_API_KEY=your-key
CLOUDINARY_API_SECRET=your-secret
ADMIN_TELEGRAM_ID=your-telegram-id

Edit file: client/.env
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Replace these values:

REACT_APP_CLOUDINARY_CLOUD_NAME=your-cloud-name
REACT_APP_CLOUDINARY_UPLOAD_PRESET=bingo_deposits
REACT_APP_ADMIN_ID=your-telegram-id


STEP 3: Deploy Database & Start App (3 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A) Deploy Database Schema:
   mysql -h your-host.aivencloud.com -u avnadmin -p < server/database/schema.sql

B) Start the App:
   Double-click: start-dev.bat

   OR manually:
   Terminal 1: npm start
   Terminal 2: cd client && npm start

C) Test:
   Backend: http://localhost:3001/health
   Frontend: http://localhost:3000
